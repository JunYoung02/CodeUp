# 1056
두 가지의 참(1) 또는 거짓(0)이 입력될 때,

참/거짓이 서로 다를 때에만 참을 출력하는 프로그램을 작성해보자.

```c
#include<stdio.h>

int main(void)
{
	int num1, num2;

	scanf("%d %d", &num1, &num2);
		// scanf_s("%d", &num1, &num2);
	printf("%d", (num1&&!num2) || (!num1&&num2));
	return 0;
}
```
서로 참 거짓이 다를 때 참을 출력하는 논리 연산을 XOR 연산이라고 부른다.

((num1) and (not num2)) or ((not num1) and (num2)) 라고 작성하면 XOR 연산을 할 수 있다.
