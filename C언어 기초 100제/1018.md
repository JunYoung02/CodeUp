# 1018

어떤 형식에 맞추어 시간이 입력될 때, 그대로 출력하는 연습을 해보자.

참고

scanf( ) 함수는 지정한 형식(format)에 따라 입력받는 함수이다.
따라서, 입력받고자 하는 형식으로 표현해주면 된다.

입력 예시   

3:16

출력 예시

3:16

<pre><code>
#include<stdio.h>

int main()
{
	int hour, min;
	scanf("%d:%d", &hour, &min);	 //scanf_s("%d:%d", &hour, &min);
	printf("%d:%d", hour, min);
	return 0;
}
</code></pre>

scanf()에서 ""(큰따옴표)안에 :(콜론)을 넣어줬기 때문에 입력할 때도 똑같이 :을 넣어서 입력한다
