두 개의 참(1) 또는 거짓(0)이 입력될 때,

모두 거짓일 때에만 참이 계산되는 프로그램을 작성해보자.
```c
#include<stdio.h>

int main(void)
{
	int num1, num2;

	scanf("%d %d", &num1, &num2);
		// scanf_s("%d", &num1, &num2);
	printf("%d", !num1 && !num2);
	return 0;
}
```
모두 거짓일 경우만 참이므로 (NOT num1 AND NOT num2)인 경우이다.

따라서 (!num1 && !num2)로 써줄 수 있다.
