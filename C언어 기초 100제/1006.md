이번에는 특수문자 출력에 도전하자!!

다음 문장을 출력하시오.

"!@#$%^&*()"
(단, 큰따옴표도 함께 출력한다.)

<pre><code>
#include<stdio.h>

int main(void)
{
	printf("\"!@#$%%^&*()\"");
	return 0;
}
</code></pre>

printf()문 안에서 큰 따옴표를 출력하기 위해 \"를 사용해주었다. 

 또한, 퍼센트 문자인 % 기호가 앞에 붙는 문자를 형식 지정자라고 하며, 그 위치에서 지정한 형식으로 값을 출력해준다.

따라서 %를 출력하기 위해서 %%로 출력해 주어야 한다.
