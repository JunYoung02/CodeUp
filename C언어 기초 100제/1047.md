# 1047

정수 1개를 입력받아 2배 곱해 출력해보자.

```c
#include<stdio.h>

int main(void)
{
	int num1;

	scanf("%d", &num1);
		// scanf_s("%d", &num1);

  printf("%d", num1 << 1);
	  // printf("%d", num1 * 2);
	return 0;
}
```
이 문제는 비트시프트 연산 문제이다.

비트시프트 연산은 2진수 형태로 저장된 값을 왼쪽(<<)이나 오른쪽(>>)으로 지정한 비트 수만큼 밀어주면 2배씩 늘거나 반으로 줄어든다.

예시로 '1010'이라는 2진수가 있다면 비트시트프 연산자로'10100' 또는 '0101'로 바꿀 수 있다.
