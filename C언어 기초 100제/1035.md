# 1035

16진수로 입력된 정수 1개를 8진수로 바꾸어 출력해보자.

```c
#include<stdio.h>

int main(void)
{
	int n;

	scanf("%x", &n);		//소문자로 16진수 입력
		//scanf_s("%x", &n);

	printf("%o", n);
	return 0;
}
```
16진수로 입력받기 위해 scanf_s()에서 %x로 입력 받았다.

이때 %x를 소문자로 입력하면 소문자로 16진수를 입력해야 하고, %X와 같이 대문자로 입력하면 대문자로 16진수를 입력해야 한다.

마지막으로 8진수로 출력하기 위해 %o를 사용해주었다.
