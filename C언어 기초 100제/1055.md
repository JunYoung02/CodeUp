# 1055
두 개의 참(1) 또는 거짓(0)이 입력될 때,

하나라도 참이면 참을 출력하는 프로그램을 작성해보자.
```c
#include<stdio.h>

int main(void)
{
	int num1, num2;

	scanf("%d %d", &num1, &num2);
		// scanf_s("%d", &num1, &num2);
	printf("%d", num1||num2);
	return 0;
}
```
둘 중 하나라도 참이면 참이므로 논리 연산자 중에서 or 연산자인 ||를 사용해주었다.

||는 쉬프트를 누르고 \(백슬래시)를 누르면 출력할 수 있다.
